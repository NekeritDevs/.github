## ¿Qué es NekeritDevs?

NekeritDevs es una comunidad abierta para desarrolladores. Un espacio donde compartir proyectos, resolver dudas técnicas, colaborar en código y aprender junto a otras personas que comparten la misma pasión por el desarrollo de software.

No somos un estudio ni una agencia. Somos devs ayudando a otros devs.

---

## ¿Qué encontrarás aquí?

- **Proyectos colaborativos** — código abierto impulsado por la comunidad
- **Soporte técnico** — canales especializados por tecnología y sistema operativo
- **Recursos y referencias** — documentación, snippets y guías prácticas
- **Discusiones técnicas** — arquitectura, buenas prácticas, tendencias del sector

## Contribuciones

Todos los repositorios públicos de esta organización aceptan contribuciones. Si quieres colaborar:

1. Abre un issue describiendo la mejora o bug
2. Haz un fork del repositorio
3. Crea una rama con un nombre descriptivo
4. Envía tu pull request con una descripción clara

Revisamos las contribuciones activamente. El código limpio, documentado y con sentido es siempre bienvenido.
